# HRMS Portal

Welcome to the HRMS Portal!

## Overview
This is a Human Resource Management System containing a client application (built with React/Vite) and a backend server.

## Getting Started

### Prerequisites
- Node.js

### Client Setup
Navigate to the `client` directory:
```bash
cd client
npm install
npm run dev
```

### Server Setup
Navigate to the `server` directory:
```bash
cd server
npm install
npm start
```
